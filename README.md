# 作品練習：部落格

作品網址：https://blogslist.herokuapp.com/

## 說明：

簡易部落個網站，具備新增文章、修改文章、刪除文章以及點讚功能。<br>

## 使用技術：

### 前端:
Create-React-App 建立環境，使用Hooks<br>
串接接後端 API<br>

### 後端：
用Node.js 框架 Express 開RESTful API串接<br>
資料庫使用PostgreSQL<br>
Postman做確認api是否成功。<br>
